Generates a strongly connected, directed graph.

Adds V vertices and E edges, to graph. Source and target vertices of each edge are randomly choosen. If self_edges is false, then no edge will have the same source and targets. When an edge is added the reverse edge is added as well, effectively doubling the number of edges in the graph. The reversed/transposed edges are marked. I.e. they become part of the transposed edge set . The non-transposed edges are marked as well and become part of the edge set . The edge set of the whole graph is thus . Additionally the reversed edge set is recorded in the respective property map for subsequent use in algorithms.
