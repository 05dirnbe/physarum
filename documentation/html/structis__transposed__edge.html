Detailed Description

template<typename EdgeTransposedMap>
struct is_transposed_edge< EdgeTransposedMap >

Custom filter class that can be used as a filter to destinguish Transposed edges from non-transposed edges.

This class can be used as a filter to create filtered sub-views of another graph. The primary idea is to use filters to distinguish between the set of edges of the whole graph , the subgraph of transposed edges and the subgraph of non-transposed edges . Note that,

holds as an invariant troughout the program. The use of filtered views helps to eliminate the edge set we are not interested to work with. A filtered graph can modify the contents of the unfiltered graph via references. Iteration is realized via MultipassIterators.
