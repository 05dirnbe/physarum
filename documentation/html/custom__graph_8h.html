Detailed Description

Configuration file containing important graph specific controls.

Author:
Michael Dirnberger <mtd@mpi-inf.mpg.de>
Version:
1.0

This header file defines the type of graph that is used by the main program. Edge and vertex properties are included from their respective headers. In addition, a custom edge-filtered graph definition is provided. A number of often used typedefs intended for use in main.cpp is given.
